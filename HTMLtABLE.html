Answer of qno 4

  1. "html"
    2. Defines the root of an HTML document. All other HTML tags should be inside the tag.
  2. Head
    3. Contains metadata about the HTML document, such as title, character encoding, and linked resources (e.g., stylesheets, scripts)
  3. title
    4. Sets the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. body
    5. Contains the main content of the HTML document that is displayed on the web page.
  5. Heading
    6. These tags are used for headings, with h1 being the highest level and h6 the lowest.
  6. p(paragraph)
    7. Defines paragraph
  7. a(anchor)
    8. Defines a hyperlink, allowing navigation to other pages or resources.
  8. img (image)
    9. Embeds an image in the document. It is a self-closing tag and requires the src attribute to specify the image source.
  9. div
    10. A container element used to group other elements together for styling or layout purposes.
  10. span
    11. Used for inline grouping of content, usually for styling specific parts of text.
  11. ul
    12. Defines an unordered list, typically used for bulleted lists.
  12. ol
    13. Defines an ordered list, typically used for numbered lists.
  13. li
    14. Defines list items
  14. table
    15. Defines a table. It is used to organize content in rows and columns.
  15. tr
    16. Defines a row in a table.
  16. td
    17. Defines a cell in a table, containing data.
  17. th
    18. defines header of cell in table
  18. form
    19. Defines an HTML form for user input, such as text fields, radio buttons, and submit buttons.
  19. input
    20. Defines an input field, allowing users to enter data. It is commonly used inside forms.
  20. button
    21. Defines a clickable button, often used to submit a form or trigger an action.
  21. br
    22. Inserts a line break in the content, often used to break text into separate lines.
  22. hr
    23. Inserts a horizontal rule (line) to separate content visually.
  23. strong
    24. Defines important text, typically displayed in bold.
  24. nav
    25. Defines a navigation section, typically containing links to other pages or sections of the website.

Answer of Qno 6

  1. HTML

    Tech Stack Name Primary Use Cases Key Features/Benefits
    HTML (HyperText Markup Language) Structuring web pages and content.
    • Defines the page structure.
    • Supports multimedia elements (images, videos, audio).
    • Semantic elements improve accessibility and SEO.

  2. CSS

    Tech Stack Name Primary Use Cases Key Features/Benefits
    CSS (Cascading Style Sheets) Styling and designing web pages.
    • Controls layout and visual appearance.
    • Supports animations and transitions.
    • Enables responsive design with media queries.

  3. JavaScript

    Tech Stack Name Primary Use Cases Key Features/Benefits
    JavaScript Adding interactivity and functionality to web pages.
    • Enables dynamic content updates.
    • Supports event-driven programming.
    • Works on both frontend (browser) and backend (Node.js).
